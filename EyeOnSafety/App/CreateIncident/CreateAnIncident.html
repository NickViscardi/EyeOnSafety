About This Map

Using this map, you can report different types of location-specific incidents. With each incident reported, our forecasting model grows in accuracy.

EyeonSafety is committed to the safety of citizens. For Emergency unsafe events please contact 911 prior to using EyeonSafety. This application is not connected in real time to first responders and should not be used for that purpose. The unsafe events reported here are expressly used to inform our Predictive Analsysis Forecast.

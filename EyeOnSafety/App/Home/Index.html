EyeOnSafety

EyeonSafety is an interactive web application that allows users to visualize
historical vehicle collision and crime data as provided by Pennsylvania's Open Data Portal.
Proactive citizens can make informed decisions in order to reduce the chances that they will experience an unsafe event.

The application can also be used to report an unsafe event in real time.

EyeonSafety also provides a projection of unsafe events, such as crime and traffic accidents.

To see how it all works, look at our map below.

About This Map

Using this map, you can analyze historical crash data for the entire state of Pennsylvania as well as historical crime data for the greater Philadelphia region. By clicking the filter button located in the upper right-hand corner of the map interface, you may filter the data that appears on the map by a variety of factors—changing the appearance of the heatmaps in real time.

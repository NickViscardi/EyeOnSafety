Predictive Analysis Forecasting



Through the use of Azure Machine Learning tools, EyeOnSafety is able to generate forecasting models that represent future crash, crime, and generally unsafe incidents.
The blue line in the above graph represents actual events that have occurred. Using that event history, a forecast model was generated to predict the event trend into the future. That forecast is represented by the red line.
